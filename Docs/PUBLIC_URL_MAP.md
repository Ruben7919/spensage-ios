## Publication note

Before external review, make sure at least the following URLs return live public content over HTTPS:

- `https://spendsage.ai`
- `https://spendsage.ai/support`
- `https://legal.spendsage.ai/privacy`
- `https://legal.spendsage.ai/terms`

The in-app legal center can continue to use bundle content, but App Store Connect still expects live public URLs for support and privacy policy fields.
