# Batch 3 · Recomendaciones

## Secuencia sugerida

1. `Inicio con control`
2. `Cuentas sin hojas`
3. `Facturas sin sustos`
4. `Reglas que ayudan`

## Enfoque creativo

- Empieza con una frustracion real y remata con una promesa concreta.
- Mantiene el tono creator, pero menos meme y mas claridad util.
- Usa punch-in sobre cifras, bloques y etiquetas para que el UI no se sienta estatico.
- Cierra cada pieza con una frase compartible, no con texto corporativo.

## Recomendacion por pieza

- `Inicio`: pieza de retencion, ideal para comunicar orden y regreso diario.
- `Cuentas`: pieza de claridad financiera, buena para Reels explicativos.
- `Facturas`: pieza de ansiedad financiera resuelta, fuerte para awareness.
- `Reglas`: pieza de automatizacion suave, util para diferenciar la app de una hoja manual.
