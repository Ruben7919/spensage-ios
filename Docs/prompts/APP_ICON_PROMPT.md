# SpendSage App Icon Prompt

Last updated: 2026-04-04

## Recommendation

For the iOS app icon, use a single mascot, not the trio. At small sizes, one character reads better and feels more premium.

Recommended lead:
- `Tikki` or `Ludo`

Recommended savings object:
- glowing piggy bank
- coin with sparkle
- tiny wallet with a gold star

## Primary Prompt

```text
Design a premium iOS app icon for SpendSage, a cute but powerful savings and spending coach app. Use one mascot only for maximum legibility at small sizes. Recommended version: Ludo, the warm striped cat guide, centered and holding or leaning against a glowing savings symbol such as a piggy bank, coin, or tiny wallet. The icon should feel polished, memorable, and trustworthy, not childish or noisy. Use a bright teal, mint, cream, and soft gold palette with strong contrast, rounded shapes, soft glow, and crisp silhouette readability. The face must stay expressive and adorable, with large eyes and a confident, helpful emotion. No text, no UI panels, no clutter, no background scenery. Compose it as a clean square icon with safe margins, premium App Store quality, and strong recognition at tiny sizes.
```

## Alternate Prompt

```text
Design a premium iOS app icon for SpendSage using Tikki, the orange cat mascot, as the only character. Tikki should be centered with a savings object that immediately communicates money control and growth, such as a glowing coin, piggy bank, or wallet charm. The style should be cute, polished, and high-end, with rounded forms, subtle shine, teal and cream accents, and warm gold highlights. The icon must read clearly at small size and feel like a modern B2C finance app with game-like charm. No text, no multi-character scene, no dashboard panels, no complex background.
```

## Negative Constraints

- Do not use all three mascots in the icon.
- Do not include text or logo wordmark.
- Do not add dense finance UI or charts.
- Do not use flat corporate banking style.
- Do not let the savings object overpower the mascot face.

## Export Notes

- Square master, high resolution.
- Leave safe margins for iOS icon masking.
- Prioritize face readability and silhouette clarity over detail.

## Xcode 26 Variant Notes

- Deliver three masters into `AppIcon.appiconset`: `Any`, `Dark`, and `Tinted`.
- `Any`: bright premium default icon.
- `Dark`: deeper background and warmer highlights so the icon still reads in dark home screen styles.
- `Tinted`: simplified high-contrast version with fewer internal colors and a stronger silhouette.
